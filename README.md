# Влияние думскроллинга на психоэмоциональное состояние студентов

Статистический анализ опроса 73 студентов на предмет распространённости думскроллинга
(компульсивного потребления негативных новостей), его связи с самооценкой и различий
между академическими группами. Проект построен на собственном собранном датасете
(Google Forms), а не на учебном/синтетическом наборе данных.

## Задача

С точки зрения продукта (медиа, соцсеть, edtech-платформа) важно понять:
- насколько массово распространён думскроллинг и нужно ли вообще на него реагировать;
- зависит ли он от пола аудитории (нужна ли сегментация коммуникации);
- совпадает ли самоощущение пользователей с их реальным поведением;
- действительно ли явление субъективно ощущается интенсивнее у тех, кто его осознаёт;
- есть ли группы (по нагрузке/курсу обучения) с повышенным риском.

## Данные

- 73 анкеты, 42 вопроса: демография, привычки потребления новостей, эмоциональные
  реакции, барьеры отказа от чтения, источники информации.
- Источник: собственный опрос через Google Forms.
- Сырые данные: [`data/survey_raw.xlsx`](data/survey_raw.xlsx).

## Методология

В анализе используются непараметрические методы (данные порядковые, нормальность не
предполагается). Для каждого теста приводится размер эффекта, а ключевой результат
дополнительно проверен на устойчивость.

| № | Метод | Вопрос | Результат | Размер эффекта |
|---|-------|--------|-----------|----------------|
| 1 | χ² критерий согласия Пирсона | Равномерно ли распределена осведомлённость? | χ² = 8.041, p = 0.045 | — |
| 2 | Точный тест Фишера | Связана ли осведомлённость с полом? | OR = 0.545, p = 0.327 | Cramér's V = 0.163 |
| 3 | Критерий Макнемара (+ точный) | Совпадает ли самооценка с поведением? | p = 0.700 / 0.701 | — |
| 4 | Ранговый критерий Манна-Уитни | Выше ли «захват» у осознающих думскроллинг? | U = 960, p < 0.001 | r = 0.475 |
| 5 | Краскел-Уоллис (+ чувствительность) | Различается ли время в новостях по курсу? | H = 17.160, p = 0.002 | ε² = 0.238 |

Методологические решения, принятые осознанно:
- по полу применён **точный тест Фишера**, а не χ²: ожидаемые частоты < 5 нарушают
  условие Кокрена для χ²-аппроксимации;
- для Макнемара приведена **точная биномиальная** версия (число дискордантных пар мало);
- метод сравнения «захвата» назван **критерием Манна-Уитни**: без цензурированных данных
  обобщённый критерий Прентиса сводится к нему;
- результат Краскела-Уоллиса проверен **анализом чувствительности** (удаление малых групп,
  огрубление категорий) — значимость сохраняется во всех сценариях (p от 0.0003 до 0.0045).

Полный код, графики и интерпретация — в [`notebooks/doomscrolling_analysis.ipynb`](notebooks/doomscrolling_analysis.ipynb).

## Ключевые выводы

1. Думскроллинг — не маргинальный феномен: почти 60% студентов знакомы с термином и
   практикуют соответствующее поведение.
2. Осведомлённость не зависит от пола (Фишер, p = 0.327) — аудитория для интервенций
   широкая, без гендерной сегментации.
3. Самооценка в целом адекватна поведению: доли «осведомлён» и «откладывал дела»
   статистически не различаются.
4. Субъективный «захват» лентой выше у тех, кто идентифицирует у себя думскроллинг
   (эффект средней силы, r = 0.475) — термин валиден.
5. Время потребления новостей значимо и **устойчиво** различается по курсу обучения
   (подтверждено анализом чувствительности), что может быть связано с учебной нагрузкой.

## Ограничения

- **Выборка:** convenience sample со смещением к 3–4 курсу бакалавриата и к женщинам;
  две группы по курсу очень малы. Выводы не обобщаются на всю совокупность без проверки.
- **Измерения:** все переменные самоотчётные, без объективных метрик экранного времени.
- **Статистика:** без поправки на множественные сравнения (ключевые эффекты её бы
  пережили, пограничный тест согласия — под вопросом); связи корреляционные, не причинные.
